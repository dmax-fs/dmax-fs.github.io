Table Tags

Tag Name Description
<table>TableWrapper element for all HTML tables.
<thead>Table HeadThe set of rows defining the column headers in a table.
<tbody>Table BodyThe set of rows containing actual table data.
<tr>Table RowTable Row container.
<td>Table DataTable Data container.
<tfoot>Table FooterThe set of rows defining the footer in a table.
