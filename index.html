HTML & CSS tags cheatsheet

Elements & Structure

Tag Name Description
<h1>HeadingThis is used for the biggest heading. They can range from h1 to h6.
<li>List elementThis can be used to make either an Unordered-list <ul> or an Ordered-list <ol>
<video>Video elementThis can be used to embed a media player, for video playback <src> will contain the
URL for the video while the <controls> will display video controls for the media player.
<div>Division elementThe <div> element is used divide an HTML document into sections Usually used
to group together items of the same section.
<td>Table DataTable Data container.
<tfoot>Table FooterThe set of rows defining the footer in a table.
